# Esercizi svolti in C++
Apprendimento progressivo e mediante esempi
Utile per il potenziamento

Come testare gli esercizi: partire dal blocco

```
#include <iostream>
using namespace std;

int main() 
{

}
```

e innestare il codice all'interno del blocco main(). Per testare usa Visual Studio oppure https://www.onlinegdb.com/

Suggerimento: provare a modificare il codice a mano, anche sbagliando, per rendersi conto di come usare la sintassi.
per ogni esercizio cambia almeno una variabile e mettici una valore diverso (per esempio un altro numero, o una lettera al posto del numero): che cosa succede? 

Osserva, prendi appunti e parliamone a lezione.
